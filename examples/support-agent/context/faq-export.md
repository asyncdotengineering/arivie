# Exporting your data

- **Self-serve export:** **Settings → Data → Export** produces a ZIP of your workspace content (JSON + attachments). Large workspaces are emailed a download link when ready.
- **Deletion:** **Settings → Data → Delete workspace** permanently removes all data after a 7-day grace period.

Export is available to **workspace owners**. For a GDPR data-subject request, escalate to support.
