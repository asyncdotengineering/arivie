## When to use this

Use **Hono** when you want a lightweight Node HTTP server (API-only backend, edge-adjacent services, or a custom framework stack) without pulling in Next.js. Mount `arivie.handler` on your routes and reuse the same `arivie.config.ts` + analytics plugin as the Next.js reference.

## Code snippet

```ts
import { Hono } from "hono";
import { serve } from "@hono/node-server";
import { arivie } from "./arivie.config.js";

const app = new Hono();

// Built-in POST /api/chat + durable session routes
app.all("*", (c) => arivie.handler(c.req.raw));

serve({ fetch: app.fetch, port: 3000 });
```

## Run it

```bash
curl -X POST http://localhost:3000/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"messages":[{"role":"user","content":"How many customers?"}]}'
```

For a full Next.js + `ArivieChat` reference, see [`examples/with-nextjs/`](/recipes/nextjs/).
