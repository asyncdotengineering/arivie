Use **Better Auth** when you want a self-hosted auth server with email/password and need `userId` from the Better Auth session for `ArivieChat` and `resolveUser`.

## Setup

See `examples/with-arivie-chat/` — copy `.env.example`, set `DATABASE_URL`, `BETTER_AUTH_SECRET`, and model keys.

```bash
pnpm add better-auth @arivie/react
```
