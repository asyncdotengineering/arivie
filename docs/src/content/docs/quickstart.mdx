Get a **streamed answer** in under 10 minutes. The fast path is the CLI — no web server, no UI wiring.

## Prerequisites

- Node.js 20+ and [pnpm](https://pnpm.io/)
- Postgres 14+ (local or Docker) when using the analytics plugin
- A model provider key (`OPENAI_API_KEY`, `GOOGLE_GENERATIVE_AI_API_KEY`, or `ANTHROPIC_API_KEY`)

## One-command peak (CLI)

```bash
pnpm dlx @arivie/cli init my-agent && cd my-agent
```

Put `OPENAI_API_KEY` and `DATABASE_URL` in `.env` (auto-loaded), then:

```bash
pnpm exec arivie chat --config arivie.config.ts
```

Pick a past conversation or start a new one, ask a question, and watch the agent stream the answer.

## Calling the agent

One-shot — returns the terminal answer string:

```ts
const text = await arivie.prompt({
  agent: "analyst",
  prompt: "How many customers did we get last week?",
  user: { userId: "u1", permissions: ["analytics:read"], dbRole: "arivie_reader" },
});
```

Streaming or full control — durable session with cursor-replayable `ArivieEvent` stream:

```ts
const handle = await arivie.sessions.create({
  agent: "analyst",
  prompt: "How many customers did we get last week?",
  user: { userId: "u1", permissions: ["analytics:read"], dbRole: "arivie_reader" },
  session: { id: "conv-123" },
});

for await (const event of handle.stream) {
  // model.delta, tool.call.started, run.completed, …
}
```

For an ephemeral, non-durable call, drop to Mastra's `agent.generate()`. See [ADR 0002 — Arivie vs Mastra ownership](/adr/0002-arivie-vs-mastra-ownership/).

## Production-shaped path (Next.js)

Use this when you need an App Router app, HTTP routes, and a web chat UI.

### 1. Create a Next.js app

```bash
pnpm create next-app my-arivie-app --typescript --app --eslint
cd my-arivie-app
```

### 2. Install Arivie packages

```bash
pnpm add @arivie/core @arivie/plugin-analytics @arivie/plugin-postgres @arivie/react
pnpm add -D @arivie/cli
```

### 3. Initialize the project

```bash
pnpm dlx @arivie/cli init
```

This scaffolds `arivie.config.ts`, semantic directory stubs, and CLI wiring.

### 4. Configure environment and database

Copy the example env template and set `DATABASE_URL`:

<Code code={envExample} lang="bash" title=".env.local" />

```bash
pnpm dlx arivie setup
```

`setup` creates the read-only DB role and runs storage migrations.

Seed the dogfood schema (matches `examples/with-nextjs/seed.sql`):

```bash
psql "$DATABASE_URL" -f path/to/seed.sql
```

### 5. Set a model provider key

Add to `.env.local`:

```bash
GOOGLE_GENERATIVE_AI_API_KEY=your-key
GOOGLE_MODEL=gemini-2.5-flash
```

Preference order in the reference config: **Gemini → Anthropic → OpenAI → mock**.
